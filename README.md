# Los Imperios de la Fórmula 1
### Análisis del dominio competitivo y las eras tecnológicas del campeonato (1950–2025)

Este proyecto analiza cómo surgen, se consolidan y colapsan los periodos de dominio en la Fórmula 1 a lo largo de su historia.  
A través de datos históricos de campeonatos de constructores y pilotos, se busca identificar patrones de supremacía competitiva y relacionarlos con cambios tecnológicos y regulatorios del deporte.

El objetivo principal es construir una visualización interactiva que permita explorar cómo distintas escuderías lograron convertir ventajas tecnológicas en dominio competitivo.

# Pregunta de investigación

¿Cómo emergen, se consolidan y colapsan los imperios tecnológicos en la Fórmula 1?

Este proyecto busca identificar si existen patrones que expliquen:

- la duración del dominio competitivo
- los momentos en que cambia el poder entre escuderías
- el impacto de cambios tecnológicos en el campeonato

---

# Datos utilizados

El análisis utiliza datos históricos del campeonato mundial de Fórmula 1 entre **1950 y 2025**, incluyendo:

- resultados de carreras
- puntos por temporada
- campeonatos de constructores
- campeonatos de pilotos

Estos datos fueron utilizados para construir métricas de dominancia competitiva.

---

# Proceso de modelado de datos

La preparación del dataset incluyó varias etapas:

### 1. Extracción de datos
Se obtuvieron registros históricos de resultados del campeonato de Fórmula 1 a través de datasets públicos y APIs.

### 2. Limpieza de datos
Se realizaron procesos de limpieza para:

- eliminar inconsistencias entre temporadas
- estandarizar nombres de constructores
- validar registros incompletos

### 3. Estructuración del dataset
Los datos fueron organizados en un formato que permitiera análisis temporal por temporada.

### 4. Construcción de métricas
Se calcularon variables clave para el análisis:

- diferencia de puntos entre 1° y 2° lugar
- porcentaje de victorias del campeón
- duración de periodos de dominancia
- número de equipos ganadores por temporada
